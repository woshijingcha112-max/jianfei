# 常用命令

##### 0.进入项目目录 

```
cd /d D:\source\sccp\codex\demo
```

**1. 重新编译 APK**

```
android\gradlew.bat :app:assembleDebug 
```

**2. 启动模拟器**

```
tools\android-sdk\emulator\emulator.exe -avd DietRecordPixel -gpu swiftshader -no-snapshot-load
```

如果你想查本机有哪些模拟器：

```
tools\android-sdk\cmdline-tools\latest\bin\avdmanager.bat list avd
```

**3. 检查模拟器是否在线**

```
tools\android-sdk\platform-tools\adb.exe devices 
```

**4. 重新安装 APK**

```
tools\android-sdk\platform-tools\adb.exe install -r android\app\build\outputs\apk\debug\app-debug.apk 
```

**5. 启动 App**

```
tools\android-sdk\platform-tools\adb.exe shell am start -n com.dietrecord.app/.MainActivity 
```

**6. 强制重启 App**

```
tools\android-sdk\platform-tools\adb.exe shell am force-stop com.dietrecord.app tools\android-sdk\platform-tools\adb.exe shell am start -n com.dietrecord.app/.MainActivity 
```

**7. 卸载再重装**

```
tools\android-sdk\platform-tools\adb.exe install -r android\app\build\outputs\apk\debug\app-debug.apk
tools\android-sdk\platform-tools\adb.exe shell am force-stop com.dietrecord.app
tools\android-sdk\platform-tools\adb.exe shell am start -n com.dietrecord.app/.MainActivity· 
```

**8. 一套最常用顺序**

```
android\gradlew.bat :app:assembleDebug tools\android-sdk\platform-tools\adb.exe install -r android\app\build\outputs\apk\debug\app-debug.apk tools\android-sdk\platform-tools\adb.exe shell am force-stop com.dietrecord.app tools\android-sdk\platform-tools\adb.exe shell am start -n com.dietrecord.app/.MainActivity
```
